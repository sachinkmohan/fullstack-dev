# TypeScript Generics Resources

## Knowledge

- [TypeScript Handbook: Generics](https://www.typescriptlang.org/docs/handbook/2/generics.html)
  The official, authoritative source. Covers all generic syntax with accurate, up-to-date examples. Use for: any syntax question, double-checking constraints, understanding `keyof` with generics.

- [TypeScript Handbook: Utility Types](https://www.typescriptlang.org/docs/handbook/utility-types.html)
  Official list of built-in utility types (`Partial`, `Pick`, `Readonly`, `ReturnType`, etc). Use for: understanding what interviewers expect you to know by name, and seeing generic patterns in practice.

- [Matt Pocock — Total TypeScript (Free tutorials)](https://www.totaltypescript.com/tutorials)
  High-quality interactive exercises by a recognized TS expert. Use for: hands-on practice, especially the "TypeScript Generics" workshop which maps well to interview scenarios.

- [TypeScript Playground](https://www.typescriptlang.org/play)
  Official in-browser TS environment with instant type feedback. Use for: testing code from every lesson — no install required.

## Wisdom (Communities)

- [r/typescript](https://reddit.com/r/typescript)
  Active community, good signal-to-noise for beginner and intermediate questions. Use for: asking why something behaves unexpectedly, or for code review of your generic practice.

- [TypeScript Discord](https://discord.com/invite/typescript)
  Real-time help from experienced TS developers. Use for: quick questions between sessions.
